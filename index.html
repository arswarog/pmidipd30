PMIDIPD30 Editor

WARNING! This is a highly experimental tool for configuring the Pyle Audio PMIDIPD30 MIDI controller. Use at your own risk!

Instead of configuring individual components, this script makes the following simplifying assumptions:

  1. We only configure the behavior of the nine channel strips (knob, fader, button). Crossfader, transport constrols, encoder, and the four leftmost buttons are not affected.
  2. All four banks are configured identically, except for the MIDI channel of the nine channel strips (which will be the number of the bank). All other controls use MIDI channel 1.
  3. Knobs, faders, and buttons will be numbered contiguously, so we only specify the first value of each range.
  4. There's a single flag that determines whether the nine consecutive buttons will be toggles or momentary.

If you need more fine-grained control, feel free to modify the JavaScript file.
